# gitbees
## About
gitbees uses the `GitHub API` to show `user data` and `repos`
## Using
### 1 -  Clone this repository
  * ```shell
    git clone 
    ```
### 2 - Install _install.bat_ if you're using Windows or _install.sh_ if you're using Linux   
  * ```shell
    install.bat
    # or
    install.sh
    ```
### 3 - Run
  * To run, type:
    * ```shell
      gitbees
      ```
